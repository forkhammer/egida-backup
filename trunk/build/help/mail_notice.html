Уведомление выполнения заданий по почте

Для того, на почту приходило уведомление о выполнении задания необходимо в настройках программы прописать следующие атрибуты:

  • E-mail отправителя — адрес почтового ящика, с которого будет выполняться отправка писем;

  • SMTP-сервер — адрес SMTP-сервера, с котрого производиться отправка писем;

  • Логин — имя пользователя для доступа к почтовому ящику отправителя;

  • Пароль — пароль для доступа к почтовому ящику;

  • E-mail получателя — адрес получателя уведомления.

Затем необходимо в настройках заданий, которые нуждаются в уведомлениях, установить переключатель «Уведомлять о выполнении по почте» во вкладке Уведомления.
